MedScribe AI

Ambient AI clinical documentation — record a visit, watch the transcript build live, get a validated SOAP note in seconds.

Role: Solo full-stack + AI engineer Author: Bajrang Solunke github.com/…/medscribe-ai ▶ Demo video

The Problem

Clinicians spend an estimated 1–2 hours daily on documentation — typing SOAP notes, looking up ICD codes, summarising visits. It's the leading driver of physician burnout. Ambient AI scribes (Abridge, Suki, Nuance DAX) prove the workflow works, but they're closed enterprise products. Building an open prototype end-to-end exercises every skill needed to build modern AI applications: speech, streaming, structured generation, real-time orchestration, and a healthcare-grade UX.

Key Engineering Decisions

  • Catalog-validated ICD codes — LLM suggests, local CMS catalog confirms; hallucinated codes flagged not silently dropped, so the trust story is visible
  • Chunked HTTP + SSE over WebSocket — reuses the existing transport, no new auth scheme, simpler ops
  • Stop-restart MediaRecorder for live chunks — ~50 ms gap accepted; trade-off documented in the spec
  • OAuth + email/password fallback — Google ID tokens verified server-side via Google's JWKs; account-linking on email match
  • Brainstorming → spec → plan → commit — shipped two sub-projects this way; specs and plans tracked under docs/superpowers/

Features Built

  • JWT authentication (bcrypt) + Google OAuth sign-in
  • Browser MediaRecorder with stop-restart 4 s chunk loop
  • Live audio waveform (Web Audio API · AnalyserNode)
  • SSE streaming pipeline with reconnection + heartbeat
  • Catalog-validated ICD-10 suggestions with AI confidence meters
  • Editable SOAP note with optimistic save
  • ReportLab PDF export
  • Dashboard with stats, time-grouped sessions, search, filter, delete, retry-finalize
  • Demo banner / no-PHI guardrails

What's Out of Scope

This is a prototype, not a production clinical product. Real adoption would require HIPAA-compliant infrastructure (BAAs with cloud + AI vendors), Epic / Cerner integration, specialty-specific accuracy validation, and audit logging. Roadmap items: speaker diarisation, medical entity extraction, risk flagging, AI follow-up questions.
